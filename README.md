# Collage Python Project

A small Python project containing example scripts for basic programming concepts.

## Structure

- `1/intro.py` - introductory Python script
- `1/string_method.py` - string method examples
- `1/test.py` - test script or practice code

## Requirements

- Python 3.x

## Usage

Run a script from the project root:

```powershell
python 1\intro.py
```
